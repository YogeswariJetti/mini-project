# Student Management System       
A responsive Student Management System developed using React and Vite. This project provides a simple interface for student registration, login, and dashboard management while demonstrating modern front-end development practices.
## Live Demo 
https://mini-project-two-sepia.vercel.app/

## Features

* Student Registration
* Student Login
* Dashboard Interface
* Session Management using Local Storage
* Responsive User Interface

## Technologies Used

* React.js
* Vite
* JavaScript
* HTML5
* CSS3

## Project Purpose

This project was created to strengthen my understanding of React, component-based architecture, state management, and responsive web application development.
